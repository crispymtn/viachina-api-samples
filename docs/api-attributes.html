Order attributes

Attribute Required? Description Example
client_reference required Your unique reference for the order ABCDEFG123456, HK_DPD_2342
bag_number optional The bag identifier for this order 1, 15-A
disposition required Can either be stock or sale stock, sale
line_items required An array of line items for this order
delivery required The delivery information for this order

   

Line item attributes

Attribute Required? Description Example
qty required Quantity of items 1, 2, 426
price required The price of ONE item in cents 12 for 12 cents, 123 for 1.23€
weight required The weight of ONE item in grams 12 for 12 grams, 1200 for 1.2 kg
parcel_number required The parcel, which the item is in 1, 2, must be consecutive numbers starting at 1
taric_code required The TARIC code (10 digits) for this item 8487905910
ecommerce_url optional An URL where this item can be found on the internet http://example.org/shop/products/1234
ecommerce_reference optional A product reference from your ecommerce system 1234

   

Delivery attributes

Attribute Required? Description Example
courier_name required Courier you want to ship the order with DPD, DHL, UPS, PostNL
courier_service required Service to use DDP31, DDP2, DDU
registered required Track order? true or false
name required Customer name John Doe
company optional Customer's company name John Doe Ltd.
street required Customer's street address Example Street 42
zip_code required Customer's zip code 12345
city required Customer's city Example City
country_code required Customer's country as ISO 3166-2 code DE, GB, NL
phone Customer's phone number. Required for UPS shipment. +31477234929
